# CLAUDE.md — 宇宙採掘ゲーム(iOS放置ゲーム)プロジェクト

## このプロジェクトについて
iPhone向け縦画面の放置型宇宙採掘ゲーム。宇宙マップ(実在50天体・対数リング配置)を舞台に、有限の宇宙船を派遣して資源を採掘・持ち帰り、実在市況に連動した価格で売却する。通貨は NOVA(VND相当額基準)。企画・数値設計は完了済みで、これから実装フェーズに入る。

## 主要な確定仕様(要約)
- 収入は**持ち帰り→ホーム在庫→店で手動売却**:着艦時は鉱石をホーム在庫へ格納(どの鉱石が+何kg増えたか可視化)、プレイヤーが商品取引所で売却した時に当日市況で換金(明細「単価 × 数量 = 金額」必須)。**自動売却は序盤には無く、中盤にスキルとして解禁**(放置収入はそこで開放)。着艦即・自動の収入は序盤には存在しない
- 船は自動ループ(飛ぶ→掘る→満杯→帰る→在庫格納→再出発。売却は店で手動)。**採掘は確率抽選・単位は個**:1セッション固定回数(暫定4回)を基準5秒間隔で抽選、当たり率は目標産出個数から逆算(セッション平均=目標。例 月の鉄=2個)。セッション消化で帰還。**採掘ロボは強化で掘れる資源が増える**(ResourceTier)。天体×資源の目標・解禁順は将来xlsx化(暫定=出現順デフォルト)。**序盤は宇宙船1種**(採掘ロボ搭載の輸送船。1隻で飛行→採掘→帰還)。**専用採掘船を常駐させ輸送船がピストンするペア運用は中盤スキルで解禁**(現地ストレージ・満杯停止はこのペア運用で機能)。速度は**全船統一**(航行研究で成長、船個別の速度強化なし)
- 移動時間 = 実距離 ÷ 統一速度(線形式)。速度カーブはxlsx「速度カーブ」シート
- 資源は2階級:収入資源(高単価・軽量)/素材資源(バルク・研究建造用)
。単価は実市況連動(NOVA/kg = 当日USD/kg × 26,300、クランプ±10%/日・基準日比±50%)
- 精錬はMVPに含む(鉄・Ni・Tiの二層価格:鉱石=指数or金属×品位0.35 → 精錬=市況×回収率70〜95%)
- オフライン:移動は無制限、採掘時間のみ2時間バジェット(GEMでLv.4=8hまで延長)。復帰時「N便帰還」明細ダイアログ
- ズームは完全シームレス(単一シーングラフ+LOD3段、最大50天体前提の座標系)
- MVP範囲:5天体(地球・月・水星・火星・小惑星帯)/2船種/精錬/オフライン/GEM+IAP/チュートリアル(付録C)

## 技術スタック
- ゲーム:Unity 6(2D URP)+ C#、iOS 16+、縦画面、60fps目標
- サーバ:Python + SQLite + Flask(+ cron)。配信はJSON+Ed25519署名、将来CDN(Cloudflare R2/Pages)publish
- IAP:StoreKit2。広告なし・ガチャなしで初回提出

## 着手順(mvp_dev_plan.md 準拠)
1. **サーバ Phase 1**(独立・先行):SQLiteスキーマ→FRED fetcher(天然ガス/WTI/鉄鉱石/石炭)→手動入力Webフォーム→日次バッチ+管理用メール通知→管理コンソール。稼働開始が早いほど価格チャートの蓄積が増える

3. 以降は mvp_dev_plan.md の Phase 1〜5 の順。各マイルストーンの完了条件(遊べる状態)を満たしてから次へ
