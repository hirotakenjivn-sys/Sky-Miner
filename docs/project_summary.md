# プロジェクト総括:宇宙採掘 放置ゲーム(iOS)
最終更新:2026-07-12

## 1. プロジェクト概要
iPhone向け縦画面・基本無料の放置型宇宙採掘ゲーム。宇宙ステーションを拠点に、実在の50天体へ有限の宇宙船を派遣し、資源を採掘・持ち帰って売却する。最大の独自性は**実在市況との連動**:資源価格は現実の当日相場(LME・貴金属スポット・先物等)に連動し、通貨 NOVA はVND相当額基準。「宇宙空間そのものがメイン画面」「シームレスズーム」「持ち帰り換金」「精錬」が体験の柱。参考タイトルは Idle Planet Miner。

## 3. 決定事項(全リスト)

### ゲーム構造
- **船種**:採掘船+輸送船の2種のみ。探査船は採掘船に統合(天体開放はクレジット購入)、高速船は廃止
- **船の挙動**:自動ループ(飛ぶ→掘る→満杯→帰る→売る→再出発)。輸送船は採掘船を常駐させてピストンする増強手段
- **速度**:全船完全統一。移動時間=実距離÷共通速度S(線形式・例外なし)。Sは航行研究で成長(累計約285倍)、一時加速はブーストアイテムのみ
- **移動時間**:中盤(木星圏)まで片道10〜15分、以降段階的に延び 土星20分→天王星30分→海王星45分→**セドナ約2時間**。移動はオフライン制限の対象外(「寝る前に外縁便を出す」遊び)
- **天体**:すべて実在の50天体(惑星・衛星・準惑星・小惑星・彗星、10バンド)。架空天体なし。アンロックはクレジット(NOVA)購入

### マップ・ズーム
- 放射状・対数距離リング配置(静的)。惑星系はクラスター集約(セマンティックズーム)、同一リング内は千鳥角度配置
- ズームは完全シームレス(単一シーングラフ+LOD3段)。倍率上限=採掘ロボットが画面高1/3、下限=全天体全景。50天体前提の座標系で初期実装

### 経済(最重要の設計群)
- **通貨**:NOVA。VND相当額基準(基準日2026-07-11、1USD=26,300VNDで換算固定)
- **収入は持ち帰り換金のみ**:着艦した瞬間に当日市況で売却。自動収入は存在しない。売却は必ず明細形式「単価 × 数量 = 金額」
- **単価の常時表示**:貨物・ストレージ・資源チップに NOVA/kg+前日比を常時併記
- **市況連動**:安定公開されている指標はすべて連動 — 日次19系統(鉄鉱石・石炭・天然ガス・WTI・材木+LME非鉄7種+貴金属5種)+週次4(W・Cr・Mo・V詳細・ウラン)。クランプ:日次±10%・基準日比±50%
- **資源2階級制**:収入資源(高単価・軽量:金属・貴金属・He3・希少物質)/素材資源(バルク:水・氷・メタン等=研究建造用で売り物ではない)。外縁バンドの収入アンカーはHe3(現実の採取構想でも天王星・海王星が本命)
- **精錬(MVPに含む)**:二層価格 — 鉱石=金属価格×品位係数0.35 → 精錬=市況価格×回収率70〜95%。MVP対象は鉄(鉱石指数→銑鉄)・Ni・Ti。合金加工(鋼・ダイス鋼SKD11・ハイスSKH51・超硬)はv1.1、価格は成分の当日市況から自動計算(W高騰→超硬値上がりが数式で再現)
- **進行曲線**:天体収入 v = v0 × 1.32^開放順(v0≒2,762 NOVA/s)。ペース目標:小惑星帯=1週目、B10到達=42日、セドナ=90日
- **輸送の役割**:①素材輸送(強化・研究の現物要求=必須)②納品契約(NPCの期限付き高額オーダー、報酬≧拘束時間×最前線収入×1.5〜3。MVP後)。常設の輸送売却ボーナスは廃止(機会費用で常に不利のため)

### 地球限定資源
- 木材(材木先物連動)+石炭(Newcastle先物連動)。生物由来2種を持つ唯一の天体。用途は居住・生命維持・バイオ研究・製鉄(コークス)

### オフライン・マネタイズ
- オフライン採掘:初期2時間で停止 → Lv.1(3h・無料)→ Lv.2(4h)/Lv.3(6h)/Lv.4(8h)はGEM
- 復帰時「留守中にN便帰還・+XXX NOVA」明細ダイアログ
- GEM入手:課金/ミッション/実績/ログイン(無課金週100〜150目安)。広告はリワードのみ(v1.1)。ガチャなし

### サブコンテンツ「マーケット」
- 相場モニター(連動23系統のウォッチリスト・チャート・前日比)+金属図鑑(各金属の用途・産出天体・**プレス金型材との関係**:SKD11のCr、SKH51のW/Mo/V/Co、超硬のWC+Co等 — 開発元の専門領域を活かした独自コンテンツ)
- MVPは簡易版(ウォッチリスト+前日比)、図鑑フル版はv1.1。社員向け展開は行わない

### チュートリアル(付録C)
- 9ステップ・約5分。山場は①初売却の明細ポップ(3分以内)②精錬体験「同じ180kgが直売の約2.8倍」。通知許可はオフライン価値の提示直後。完了報酬GEM30+採掘船2隻目

### 技術・サーバ
- ゲーム:Unity 6(2D URP)/ iOS 16+ / StoreKit2。経済数値はxlsx→CSV/JSONの表駆動、balance.jsonリモート更新可
- 価格サーバ:Python+SQLite+Flask。日次バッチ(取得→±30%超は保留アラート→クランプ→NOVA換算→Ed25519署名JSON)。Phase 1=管理者用モニター(手動入力・管理用メール通知)、Phase 2=CDN publishでゲーム配信。MVPの自動取得はFRED系(公的・再配布可)のみで審査提出可能な構成
- 開発環境:iOSビルド・実機テスト・提出はMac必須(開発自体はWin/Mac可。MacBook一本化を推奨)。Apple Developer Program 年$99

## 7. 設計経緯の記録(主要な転換点)
1. 探査船→採掘船に統合、天体開放はクレジット制に(シンプル化)
2. 片道時間は一律20分上限→速度統一の要請で線形式へ→終盤は距離の重さを戻し最遠2時間に
3. 資源単価:仮の相対値→現実市場価格比率→**当日市況連動**→VND基準(NOVA)へと段階進化
4. 換金方式:全量輸送(氷天体で物理破綻)→現地自動換金(買い手不在の指摘)→ワラント方式(+20%輸送は機会費用で死に選択肢)→**持ち帰り換金**に確定。「自分で採ってきた感」を経済の中心に置く現在形が最終形
5. 精錬をv1.1予定からMVPへ繰り上げ(チュートリアルの山場「同じ180kgが2.8倍」が成立)
