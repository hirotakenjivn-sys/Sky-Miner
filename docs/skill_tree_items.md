# スキルツリー 項目一覧(構成・数値なし)

放置ゲーのスキルツリー/転生システムの**項目定義**。数値(効果量・コスト曲線・前提の重み)は別途詰める。
ここでは「どんなノードが・どの系統に・何をするか」だけを確定させる。

## コスト設計の原則(重要・2026-07-14 ユーザー確定方向)
スキル強化のコストは **NOVA だけでなく 精錬品/合金/クラフト品などの希少資源** を要求する。
生産チェーン(採掘→精錬100:1→工場)の出口をスキルに繋ぎ、資源のシンクにする。資源は意図的に不足させ、
多数の分岐スキルから「何に投資するか」を毎回迷わせる=狙った良ゲー性(memory: skill-cost-scarcity)。
→ 各ノードの正式コストは「◯◯を△個 + NOVA」の形で後で詰める(このファイルは項目のみ)。

- ラン(毎回リセット)= `[スキル]`タブ / NOVA + 希少資源 で購入 / 転生で 0 に戻る
- 恒久(転生後も保持)= `[転生]`タブ / 転生通貨で購入 / 永久
- ノードidは英小文字。将来 `art/skills/<id>.png`(SFテックエンブレム)を置けば中央リゾルバ(SpriteBank)経由で全画面反映。
- 関連メモリ:prestige-goal(転生の最終目標・Save を run/meta 分離)、upgrade-curve-milestones(効率曲線の段差)。

## ① ランツリー(NOVA・転生でリセット)

### 中央
| id | 項目 | 内容 |
|---|---|---|
| `core` | 起点 | ツリーの根。ここから4系統へ分岐 |

### 採掘系(青)
| id | 項目 | 内容 |
|---|---|---|
| `mine_yield` | 採掘効率 | 1セッションの産出個数を増やす(既存の主レバー) |
| `mine_rate` | 採掘速度 | 抽選間隔を短縮=1便が速くなる |
| `mine_luck` | 抽選確率UP | 当たり率を底上げ=外れが減る |
| `mine_rare` | レア鉱脈 | 上位の資源Tierを解禁(高単価鉱物が出るように) |

### 物流系(橙)
| id | 項目 | 内容 |
|---|---|---|
| `ship_add` | 宇宙船増設 | 稼働船を1隻増やす(上限あり) |
| `nav_speed` | 航行速度 | 全船の移動速度を上げる(速度は全船統一) |
| `pair_ops` | ペア運用解禁 | 採掘船常駐+輸送船ピストンを解禁(中盤スキル) |
| `fleet_offline` | 留守番強化 | オフライン採掘バジェットを延長 |

### 市場系(緑)
| id | 項目 | 内容 |
|---|---|---|
| `auto_sell` | 自動売却 | 「閾値単価を超えたら売る」ルール売りを解禁(放置収入の要) |
| `contract_slot` | 契約枠拡張 | 同時に受けられる納品契約(オーダー)数を増やす |
| `price_bonus` | 価格ボーナス | 売却単価に上乗せ |
| `market_forecast` | 市況予測 | 価格チャートの表示期間を延ばし、売り時判断を助ける |

### 生産系(紫)
| id | 項目 | 内容 |
|---|---|---|
| `refinery` | 精錬所解禁 | 鉱石→金属(鉄/Ni/Ti)を自動変換=在庫価値2倍 |
| `refine_rate` | 回収率UP | 精錬の変換能力/倍率を上げる |
| `factory` | 工場解禁 | 合金クラフト(鋼/真鍮/ステンレス/ダイス鋼/ハイス/超硬)を解禁 |
| `build_craft` | 建造 | 素材から船/施設を作る(例:鋼材→採掘ドローン、ソーラー→オフライン延長、充電器→採掘高速化) |

**ランツリー計:17ノード**(Core + 採掘4 + 物流4 + 市場4 + 生産4)

## ② 恒久層(転生・転生通貨・永久保持)

### (a) 前借り系 — ランノードの底上げ〔絵はランの17を色替え流用=新規生成なし〕
| id | 項目 | 内容 |
|---|---|---|
| `meta_yield` | 採掘効率の下駄 | 毎ランを採掘効率+Nレベルからスタート |
| `meta_ship_start` | 初期船数+ | 転生後の開始船数を増やす |
| `meta_start_res` | 初期解禁資源 | 鉄以外も最初から解禁 |
| `meta_start_refinery` | 精錬所を最初から | 転生後すぐ精錬所が使える |
| `meta_start_pair` | ペア運用を最初から | 転生後すぐペア運用が使える |

### (b) メタ専用系 — ランに対応がない恒久効果〔新規エンブレムが要るのはここだけ〕
| id | 項目 | 内容 |
|---|---|---|
| `asc_core` | 転生解禁 | 転生システムそのものの起点/解禁 |
| `asc_gain` | 転生通貨効率 | 1回の転生で得る通貨量を増やす |
| `asc_seed` | 初期資金 | 転生後の所持NOVAを底上げ |
| `asc_offline` | オフライン恒久延長 | 留守中の採掘時間上限を永久に延ばす |
| `asc_income` | 全収入UP | すべての収入に恒久倍率 |
| `asc_unlock` | 到達緩和 | 次の転生条件を緩和/到達ボーナス |

**恒久層計:前借り5 + メタ専用6 = 11ノード**(絵は前借り流用なので**新規生成はメタ専用6のみ**)

## 生成が必要なエンブレム枚数
| セット | 枚数 |
|---|---|
| ランツリー(青/橙/緑/紫+Core) | 17 |
| 恒久・前借り(色替え流用) | 0 |
| 恒久・メタ専用 | 6 |
| **合計** | **約23枚** |

スタイル=SFテックエンブレム(共通フレーム+中央シンボル差し替え、取得=発光ON/未取得=OFF)。
